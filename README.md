# EXP-1-PROMPT-ENGINEERING
### NAME : HARI PRIYA M
### REG NO : 212224240047
Experiment: Develop a comprehensive report for the following exercises:
- Explain the foundational concepts of Generative AI.
- Focusing on Generative AI architectures. (like transformers).
- Generative AI applications.
- Generative AI impact of scaling in LLMs.

## Aim: 
To develop a Comprehensive Report on the Fundamentals of Generative AI and Large Language Models (LLMs)

## Scenario
A student needs a well-structured report on Generative AI for academic submission. Instead of manually compiling information, the student uses prompt engineering to generate a high-quality, structured report using an AI model.

## Prompt Used
Generate a well-structured academic report on Generative AI and Large Language Models.

Include:
- Foundational concepts of Generative AI  
- Transformer architecture explanation  
- Applications of Generative AI  
- Impact of scaling in LLMs  

Requirements:
- Use clear headings and paragraphs  
- Maintain formal academic tone  
- Keep the content concise and well-aligned  
- Avoid unnecessary sections like aim or algorithm  
- Ensure the output is suitable for converting into a PDF report  

## Algorithm:
1. Define the objective of the report  
2. Design a structured prompt with clear requirements  
3. Input the prompt into a Generative AI model  
4. Generate and review the output  
5. Refine content if needed  
6. Convert the final output into PDF format
 
## Output
final_report.pdf

## Result
A clear, concise, and structured report on Generative AI was successfully generated. The experiment demonstrates that well-designed prompts significantly improve the quality, clarity, and usability of AI-generated content for academic purposes.
